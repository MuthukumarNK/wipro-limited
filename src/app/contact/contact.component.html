Angular 8 pipes

Pipes are used to Transform Data. It is denoted by |. It transforms the data in the format as required and displays the same in the browser.



UpperCase Pipe and LowerCase Pipes


{{title | uppercase}}

{{title | lowercase}}

{{'Raj' | uppercase}}


Currency Pipe

My Salary is : {{salary | currency:"INR"}}


Decimal Pipe

{{ 454.78787814 }} - {{ 454.78787814 | number:'3.4-4' }}


{{ 5.123456789 }} - {{ 5.123456789 | number:'3.1-5' }}


JSON Pipe

{{ jsonval | json }}

Slice Pipes

{{ months | slice:2:8 }}
